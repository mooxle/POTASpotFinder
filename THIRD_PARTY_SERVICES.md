# Third Party Services

This project uses the following external APIs:

## OpenStreetMap / Overpass
- Public query endpoints
- Usage must comply with Overpass API policies
- Users should consider self-hosting for heavy usage

## OpenTopoData
- Limit: 1 request/second, 1000/day

## Open-Elevation
- Free tier limited
- Users should provide their own endpoint for production use

## Disclaimer
This project is not affiliated with any of these services.
Users are responsible for complying with API terms.
